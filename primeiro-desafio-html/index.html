HTML: Estrutura e Tags para Construção de Páginas Web

O HTML foi introduzido em 1991 e evoluiu para a versão 5, que trouxe uma ampla gama de recursos e funcionalidades para aprimorar o desenvolvimento web.Sua principal função é estruturar páginas da web, utilizando elementos ou tags HTML que se interconectam para formar o conteúdo da página.

Esses elementos são essenciais para informar ao navegador como interpretar e exibir diferentes partes da página, como títulos, parágrafos, imagens, links, entre outros elementos.Para que um documento HTML seja corretamente interpretado pelo navegador, ele deve ter a extensão .html e seguir a sintaxe específica das tags HTML. As tags são formatadas com um sinal de menor que (<), seguido pelo nome do elemento e fechadas com um sinal de maior que (>). Algumas tags requerem um fechamento explícito com uma barra (/), enquanto outras são autocerrantes (self-closing) e não necessitam desse caractere.

Estrutura básica do HTML

            
                <!DOCTYPE html>
                <html lang=>
                <head>
                    <meta charset="UTF-8">
                    <title>Minha Página HTML</title>
                </head>
                <body>
                    <h1>Olá, Mundo!</h1>
                    <p>Este é um parágrafo de exemplo.</p>
                </body>
                </html>

<!DOCTYPE html>: Indica ao navegador que o documento é do tipo HTML e especifica sua versão. Quando simplesmente escrito como html, refere-se à versão mais recente do HTML.


<html>: Funciona como a raiz do documento HTML, agindo como um contêiner que envolve todos os outros elementos HTML.


<body>: Contém todo o conteúdo visível para o usuário na página, como texto, imagens e vídeos, estruturado através das demais tags HTML.


<script>: Inclui instruções de script diretamente no documento ou aponta para um arquivo de script externo usando o atributo src.


<head>: Contém informaçõe o documento interpretadas pelo navegador, como metadados, título da página e links para folhas de estilo.


<meta>: Define metadados que fornecem informaçõe o documento HTML, como o conjunto de caracteres utilizado, autor, configurações da janela de visualização, entre outros.


<title>: A tag é utilizada no elemento <head> de um documento HTML para definir o título da página web. Este título é exibido na barra de título do navegador ou na aba quando a página é carregada.



Tipos de Tags

As tags de formatação de texto em HTML são usadas para aplicar estilos e modificar a apresentação visual do texto em uma página web. Elas permitem que os desenvolvedores especifiquem como o texto deve ser exibido sem alterar o significado ou a estrutura semântica do conteúdo. Aqui estão algumas das principais tags de formatação de texto em HTML:


<b>: Define um texto em negrito.


<strong>: Define um texto que é semanticamente importante e geralmente é exibido em negrito. É usado para indicar ênfase forte, que pode ser interpretada por leitores de tela e motores de busca.


<i>: Define um texto em itálico.


<em>: Define um texto que é semanticamente enfatizado. Geralmente é exibido em itálico, mas seu propósito é fornecer ênfase adicional ao conteúdo.


<u>: Define um texto sublinhado. É menos comum hoje em dia devido a diretrizes de acessibilidade que recomendam não usar sublinhado para além de links.


<s>: Define um texto que deve ser riscado.


<del>: Define um texto que foi removido do documento. É geralmente exibido com linha através do texto.


<ins>: Define um texto que foi inserido no documento. É geralmente exibido sublinhado.


<mark>: Define um texto que foi removido do documento. É geralmente exibido com linha através do texto.


<h1> a <h6>: As tags <h1> até <h6> são usadas em HTML para definir os cabeçalhos de seção em uma página web. Elas representam diferentes níveis de importância e hierarquia do conteúdo, com <h1> sendo o mais importante e <h6> o menos importante.


Exemplos:

Este é um título principal

Este é um título de seção principal

Este é um título de subseção

Este é um título de outra subseção

Este é um título de subseção menos importante
Este é um título de subseção muito específica


<pre>: A tag <pre> em HTML é usada para representar texto pré-formatado, o que significa que o texto dentro dela é renderizado mantendo espaços em branco, quebras de linha e qualquer formatação original. É ideal para exibir blocos de código, poesia ou qualquer texto que dependa da formatação exata.


<q>: Esta tag HTML é utilizada para marcar citações curtas dentro de um texto. Ela é geralmente renderizada pelo navegador com aspas ao redor do texto citado.

<blockquote>: A tag <blockquote> em HTML utilizada para indicar uma citação em bloco dentro de um documento HTML. Ela é geralmente usada quando se deseja destacar um trecho de texto que foi retirado de outra fonte, como um livro, artigo, discurso, entre outros.



<sup>: A tag <sup> em HTML é usada para renderizar texto sobrescrito, que é geralmente utilizado para exibir expoentes ou indicadores de nota de rodapé.


<br>: A tag <br> em HTML é usada para criar uma quebra de linha dentro de um texto, sem iniciar um novo parágrafo. É uma tag vazia, o que significa que não possui tag de fechamento.


<hr>: A tag <hr> em HTML é usada para criar uma linha horizontal que separa o conteúdo dentro de um documento. É útil para dividir seções distintas ou para marcar uma mudança significativa de conteúdo.


<sub>;: A tag <sub> em HTML é usada para renderizar texto como subscrito, o que significa que o texto é exibido abaixo da linha de base normal do texto, geralmente para fórmulas químicas, notas de rodapé ou expressões matemáticas.


<code>: A tag <code> em HTML é usada para definir blocos de código ou partes de código em um documento. Ela indica que o texto contido deve ser tratado como código e é frequentemente renderizada com uma fonte monoespaçada para diferenciá-la do texto normal.


Listas Ordenadas e Não Ordenadas


Listas Ordenadas <ol>: A tag <ol> é usada para criar listas ordenadas, onde cada item da lista é numerado sequencialmente.

Exemplos: 

                <ol>
                    <li>Item 1<li>
                    <li>Item 2<li>
                    <li>Item 3<li>
                <ol>


Listas Não Ordenadas <ul>: A tag <ul> é usada para criar listas não ordenadas, onde cada item da lista é marcado com um ponto (•) por padrão, mas isso pode variar dependendo do navegador ou do estilo CSS aplicado.

Exemplos: 

                <ul>
                    <li>Item 1<li>
                    <li>Item 2<li>
                    <li>Item 3<li>
                <ul>

Links


<a>: A tag <a>é usada para criar links em HTML. Ela é essencial para navegação e interatividade em páginas web. 

            
                <a> href="https://www.example.com">Exemplo de link </a>

Atributos importantes da tag <a>:

  • <href>: Define o destino do link, que pode ser uma URL, um arquivo ou um âncora interna.
  • <target>: Define onde abrir o link (por exemplo, em uma nova janela ou na mesma janela).
  • <title>: Fornece um texto descritivo quando o usuário passa o mouse sobre o link.


Imagens


A tag <img> em HTML é usada para inserir imagens em uma página web. Ela não possui uma tag de fechamento e usa atributos para especificar detalhes sobre a imagem que será exibida. Aqui estão os principais atributos da tag <img>: 

            
                <src="caminho/para/imagem.jpg" alt="Descrição da imagem" width="300" height="200" title="Título da imagem">

Atributos importantes da tag <a>:

  • <src>: Especifica o caminho (URL) para a imagem. É um atributo obrigatório.
  • <alt>: Define um texto alternativo para a imagem, que é exibido quando a imagem não pode ser carregada ou acessada por usuários com deficiência visual. Também é importante para SEO.
  • <width> e <height>: FEspecificam a largura e a altura da imagem, respectivamente, em pixels. São opcionais, mas recomendados para melhorar o desempenho da página.
  • <title>: Fornece um texto descritivo que aparece quando o usuário passa o mouse sobre a imagem. É opcional.
